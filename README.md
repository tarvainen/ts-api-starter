# TypeScript REST API starter

This starter includes

- [x] Clear project structure
- [x] Automated API documentation
- [x] DependencyInjection
- [x] Validation
- [x] Linting
- [ ] Logging
- [ ] Rate limits
- [ ] Automation tests
- [ ] Admin dashboard

## Usage

```
# Start development server at localhost:8080
npm start

# Lint
npm run lint
```
